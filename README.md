### 🚀 Highlighted Telecom Automation Projects

*   **Kalashnikova FTTH Homepass Hybrid**
    *Automasi geospasial berbasis Python & Google Earth Engine untuk audit rute jaringan, eliminasi false-positive via OSM Overpass, dan kalkulasi vertikal gedung bertingkat.*
*   **Kalashnikova-47 MapCad GIS Generator**
    *Aplikasi Flask (Zero Heavy-GIS Dependencies) untuk konversi otomatis data peta publik (OSM) langsung menjadi koordinat lokal CAD origin (0,0) siap pakai.*
*   **HPDB KMZ to Excel Template Filler**
    *Engine otomatis pembaca hierarki spasial (FDT ➔ LINE ➔ FAT ➔ HP) untuk injeksi data instan ke template Excel pelaporan APD/ABD tanpa merusak formatting asli.*
*   **Kalashnikova KMZ Validator Super Final**
    *Sistem inspeksi satu klik untuk validasi skema FDT dinamis, pencocokan warna spasial KML, serta sinkronisasi otomatis ke Bill of Quantity (BOQ) klien.*
*   **Kalashnikova BOQ Splitter & Converter**
    *GUI Python cerdas untuk ekstraksi dan pemisahan otomatis material & service dari berkas Excel BOQ mentah untuk kebutuhan upload sistem ERP internal.*
